鹿鸣读书协会工作简章

部门职责与工作流程

部门 主要职责与目标 工作流程
社长（副社长） 1. 全面负责社团的运营和管理
2. 设定社团的愿景、使命和方向
3. 监督并协调各部门工作
4. 代表社团与外界沟通联络		 1. 设定社团年度及学期目标和计划
2. 定期召开社团全体会议，通报工作进展和决策
3. 对各部门工作进行定期检查和评估
4. 代表社团出席重要活动或会议，维护社团形象
5. 制作相关PPT 向学院老师汇报近期工作内容和未来工作计划等
文化部 1. 制定社团的活动策划案和预算
2. 安排社团活动的场地、时间和资源；安排相关活动人员
3. 监督活动进展，确保活动顺利进行
4. 提供新颖、有创意的活动设想		 1. 根据社团目标和各部门需求，制定活动计划
2. 协调人员分配及活动场地、物资等资源，确保活动顺利进行和有序收场
3. 向会长提交活动审批表、摆台申请表、活动当天工作人员表等
部门 主要职责与目标 工作流程
外联部 1. 负责社团的对外联络与合作
2. 寻求和建立外部资源合作，为社团提供支持和帮助
3. 物资保障，提交报销相关材料
4. 组织社团团建及部门间的联谊 促进交流合作		 1. 寻找和联系联谊社团与南京其他高校或本校其余社团
2. 各类物资采购，整理报销相关材料
3. 社团资金管理
4. 定期向社团汇报外联工作进展和成果
计划部 1. 细化并如期实施鹿鸣摘星计划
2. 联络对接理学院预科 大一 及大二各班班委
3. 协助参与并监督相关活动的实施安排
4. 各类表格制作		 1. 联络对接班级班委，收集鹿鸣摘星计划具体要求的读后感和跑步记录截图与活动开展。
2. 摆台表，活动当天工作人员表 ，pu签到，操行统计盖章，人员工作核定（每次活动向各部门副部长要求参与活动人员名单进行汇总记录统计，学期末根据鹿鸣规则章程评定人员是否合格以发放任职操行证明）等。
宣传部 1. 负责社团的宣传和推广工作
2. 设计并制作宣传材料，提升社团形象
3. 通过各种渠道宣传社团的活动和成果（如：撰写新闻稿发布公众号）		 1. 根据活动计划和目标，制定宣传方案
2. 设计并制作海报、传单等宣传材料
3. 通过QQ，微信公众号等渠道发布活动信息
4. 记录活动过程（拍照 录像等）
5. 收集和分析宣传效果，提出改进建议

注意事项

（以下内容中鹿鸣摘星计划简称“鹿鸣计划”）

一、各个部门间的交流对接情况：

  1. 文化部 —— 会长

    • 将初步的活动设想上交给会长；
    • 交流确定活动形式；
    • 文化部根据会长的社团学期和学年活动计划撰写活动策划案并交给会长审核；
    • 反馈总结各类活动进行中存在的问题或前后期的准备、总结工作中的不足。

  2. 文化部 —— 外联部

    • 外联部根据文化部提供的活动策划案和经费来确定奖品等活动相关物资；
    • 商定是否需要联络其他社团或高校，并反馈给会长；

  3. 文化部 —— 计划部

    • 文化部根据文化部提供的活动策划案来制作初步的工作人员安排表（根据社团成员课程表），可以根据成员反馈进行适当调整，注意提前通知；
    • 根据计划部提供的工作人员安排表督促部门成员积极参与社团活动；

  4. 文化部 —— 宣传部

    • 将活动策划案及时递交给宣传部以便撰写活动宣传文案及新闻稿等

  5. 外联部 —— 会长

    • 按学期总结账单上交给会长；
    • 定期反馈经费余额等情况；
    • 反馈各类活动进行中存在的问题或前后期的准备、总结工作中的不足。

  6. 外联部 —— 计划部

    • 根据计划部提供的工作人员安排表督促部门成员积极参与社团活动；
    • 根据鹿鸣计划具体实施情况和社团经费商定鹿鸣计划相关奖品奖项

  7. 外联部 —— 宣传部

    • 确定活动相关奖品后给宣传部提供照片等以便用于活动宣传

  8. 计划部 —— 会长

    • 定期汇报鹿鸣计划实施情况；
    • 反馈各类活动进行中存在的问题或前后期的准备、总结工作中的不足。

  9. 计划部 —— 宣传部

    • 将鹿鸣计划收集到的优秀文章/读后感递交给宣传部以便撰写活动新闻稿宣传此类社团活动；
    • 商定是否需要进行鹿鸣计划征文活动以作为定期的活动总结，反馈活动进行情况和活动效果；

  10. 宣传部 —— 会长

    • 将宣传文案和新闻稿上交给会长审核；
    • 反馈各类活动进行中存在的问题或前后期的准备、总结工作中的不足。

二、鹿鸣咖啡厅的运营

  1. 副会长负责接听接待社团及其他相关人员申请场地使用并嘱咐其活动结束后保持场地干净卫生和填写场地使用记录表
  2. 与指导老师、会长及外联商定需如何布置发展咖啡厅以实现场地充分利用，亦可用于宣传社团

三、鹿鸣成员出席活动规定、惩奖机制及退组织相关章程

  1. 出席活动

    • 组织内部活动（包括但不限于游学、团建聚餐、鹿鸣读书会、外联讲座等）不强制参加，活动会根据人员人数调整时间，个人根据时间选择是否能够参与，pu分和操行分是否发布视人数多少和具体情况而定。
    • 组织作为活动主办方时，鹿鸣全体人员听从会长和组织部人员安排，根据每次活动内容抽调部门干事，流程为：由计划部制定人员安排表→个人时间原因无法参与时自己私下联系部门其他干事调整并告知部长。每次活动由副部长记录人员参与情况，并整理汇报给计划部，参与干事获得相关活动的操行分/pu分。再一次活动将不在指定人员范围之内。
    • 组织作为活动协办方时（协助校文联等组织的活动），自愿参加，鼓励参加。视情况或获得协办方的相关福利。

  2. 奖惩机制

    • 对于认真负责、工作突出的干事，年度评选优秀干事。2024-2025年度进行工作核定，评定合格进行操行分证明发放。
    • 对于工作态度特别不认真的干事(标准具体参见备注)，鹿鸣有权劝退。
      • 标准：一学年的全会两次及以上未参加、部长团安排任务超过3次及以上拒绝且不协调沟通、大型全体工作缺席两次及以上、一心水pu分操行分对活动态度不认真者，均有权劝退。

  3. 请假及退组织相关说明

    • 请假：
      • 全会请假需按照固定请假条格式写假条，提前交给部长或者让其他干事代给。
      • 活动方面，是否参与请至少提前一天向部长说明，由部长上报给会长/副会长。
      • 沟通时说明原因，保持基本礼仪。建立理解在良好沟通上。
    • 退组织：
      • 请在决定退组织之前和部长进行积极沟通和说明。
      • 部长向会长说明人员调离情况。
      • 鹿鸣尊重每个人的选择，底线是互相不诋毁。
